# Integrações

Documente integrações externas, autenticação, payloads, schemas, erros, retries, fallback, auditoria e métricas.

## Como preencher

Crie um arquivo ou seção por integração.

Para cada integração, documente:

- sistema;
- finalidade;
- autenticação;
- permissões;
- endpoints;
- payloads;
- erros esperados;
- timeout/retry;
- fallback;
- logs e métricas.

## Exemplo

| Campo | Exemplo |
| --- | --- |
| Sistema | Ferramenta de tickets |
| Finalidade | Criar handoff humano |
| Fallback | Registrar fila local e avisar usuário |
