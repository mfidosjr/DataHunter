# Backlog MVP

Organize épicos, features, histórias, critérios de aceite e relação com DoD.

## Como preencher

Derive o backlog do CONOPS, SRS, DoD MVP e riscos.

Cada item deve ter:

- épico;
- história;
- prioridade;
- critério de aceite;
- requisito relacionado;
- teste esperado;
- status.
