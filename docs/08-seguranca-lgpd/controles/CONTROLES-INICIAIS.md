# Controles Iniciais

## Finalidade

Listar controles mínimos de segurança, privacidade e conformidade para o MVP.

## Como preencher

Para cada controle, defina objetivo, implementação, evidência e owner.

- Menor privilégio.
- Gestão de segredos.
- Minimização de dados.
- Retenção.
- Auditoria.
- Bloqueio de conteúdo sensível.
- Testes de prompt injection.
- Resposta a incidentes.

## Exemplo

| Controle | Como implementar | Evidência |
| --- | --- | --- |
| Menor privilégio | Escopos mínimos nos conectores | Revisão de permissões |
| Gestão de segredos | Variáveis seguras ou secret manager | Ausência de segredo no Git |
